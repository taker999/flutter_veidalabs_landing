# Jiji AI Landing Screen

A responsive Flutter landing screen for an AI assistant with modern UI components.

## Features

- Responsive design with `flutter_screenutil`
- Centralized color management (`AppColors`)
- Interactive search bar
- AI response card with bullet points
- Resource cards (PDF, YouTube)

## Setup

```yaml
dependencies:
  flutter_screenutil: ^5.9.0
```

```bash
flutter pub get
flutter run
```

## Customization

**Change colors**: Edit `app_colors.dart`  
**Update avatar**: Replace image URL in 'app_images.dart' 
**Modify content**: Edit text in response cards

## Design

- Design size: 375 x 812
- Fully responsive across all devices
